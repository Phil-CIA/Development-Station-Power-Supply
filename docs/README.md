# docs

Design documentation, datasheets, and reference material for the Development Station Power Supply project.

## Suggested Contents

| Item | Description |
|---|---|
| Datasheets | PDF datasheets for key ICs (ST7796S, XPT2046, ESP32-C6, LDO/switcher, etc.) |
| Design notes | Markdown or PDF notes on design decisions (filter calculations, BOM choices, etc.) |
| Connector pinouts | Tables or diagrams for the inter-board ribbon/connector |
| Test plans | Procedures for bring-up, calibration, and acceptance testing |

Add files here as the project progresses. Keep datasheet PDFs named clearly, e.g., `ST7796S_datasheet.pdf`.
